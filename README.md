# Technical Development Stack

Engineered for precision, performance, and minimalism.

## Core Stack

- **Framework:** Next.js (Static Site Generation)
- **Development Environment:** Windows + Node.js
- **Content Format:** Markdown
- **Styling:** Minimal CSS / Tailwind CSS
- **Version Control:** Git + GitHub
- **CI/CD:** GitHub Actions (Automated SSH Deployment)

---

# Architecture Overview

Local Development (Windows + Node.js)  
→ GitHub Repository  
→ GitHub Actions (CI/CD)  
→ Ubuntu VPS (Hestia)  
→ Nginx serving static export  

### Production Characteristics

- Static HTML output
- No database
- No runtime Node server
- No dynamic backend
- Lightweight infrastructure
- Maximum SEO potential
